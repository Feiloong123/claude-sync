When organizing academic literature, follow this workflow:
1. Create a Word (.docx) summary of all references with DOIs and methodology tables
2. Download PDFs for all references into categorized folders under `papers/<category>/`
3. PDF naming: `年份-期刊缩写-题目关键词.pdf` (e.g., `2004-J_Membr_Sci-Crosslinked_PVA_membranes.pdf`)
4. Verify every PDF is valid (magic bytes `%PDF-`, file size > 50KB, can be opened)
5. For OA papers: try publisher direct first; for paywalled: use Sci-Hub

**Why:** User wants reproducible literature management; without DOI/PDF verification, references are unreliable.

**How to apply:** Every time research content with references is produced, immediately follow up with PDF download and organization. Never leave references without corresponding PDFs.
